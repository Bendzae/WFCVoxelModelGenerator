# 3D Voxel WFC Implementation & UI Client

Heavy WIP, currently only supports 2-dimensional voxel grids

Uses JavaFX for UI and 3D-Rendering

## Prerequisites
 * Maven
 * Java 11+

## How to run
```
git clone https://git.haw-hamburg.de/acg303/wfc
```
```
cd /wfc
```
```
mvn install
```

Run App.Java
